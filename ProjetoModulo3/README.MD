## 📌 Desafio Escolhido

O grupo desenvolveu um aplicativo simples de organização de tarefas chamado **TaskFlow AI**

O objetivo do projeto foi criar uma aplicação funcional capaz de auxiliar usuários no gerenciamento de tarefas diárias, utilizando recursos visuais Low Code/No Code aliados ao suporte estratégico de Inteligência Artificial (Vibecode).

O sistema permite:

* Cadastro e login de usuários
* Criação, edição e exclusão de tarefas
* Definição de prioridades
* Organização por categorias
* Controle de tarefas concluídas e pendentes
* Sugestões automáticas de tarefas utilizando IA

## 📚 Reflexão Crítica

O grupo percebeu que as plataformas Low Code/No Code aceleram bastante o desenvolvimento inicial, porém apresentam limitações em personalizações mais avançadas.

Para contornar essas dificuldades, foram utilizadas soluções criativas como:

* simplificação do escopo do aplicativo;
* utilização de plugins prontos;
* organização modular das funcionalidades;
* foco em experiência do usuário e usabilidade.

A Inteligência Artificial foi utilizada como apoio estratégico para geração de ideias, organização de funcionalidades e melhoria do fluxo da aplicação.
