# Engenharia de Prompt e Aplicações em IA

Este repositório reúne materiais, exemplos, exercícios e projetos desenvolvidos na disciplina **Engenharia de Prompt e Aplicações em Inteligência Artificial**, parte do curso de **Ciência da Computação**.

## 🎯 Objetivos da disciplina
- Introduzir conceitos fundamentais de **Engenharia de Prompt**  
- Explorar técnicas para projetar interações eficazes com modelos de IA  
- Desenvolver aplicações práticas que utilizam **IA generativa** em diferentes contextos  
- Estimular o pensamento crítico sobre os impactos sociais, éticos e tecnológicos da IA  

## 📂 Estrutura do repositório
- **/aulas** → materiais de apoio e exemplos apresentados em sala  
- **/exercicios** → atividades práticas para fixação dos conteúdos  
- **/projetos** → trabalhos desenvolvidos pelos alunos ao longo da disciplina
